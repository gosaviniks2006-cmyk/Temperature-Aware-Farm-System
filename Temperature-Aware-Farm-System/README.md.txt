Temperature-Aware Farm System

Project Overview

This project uses Artificial Intelligence and Machine Learning techniques to predict spoilage risk in agricultural produce based on environmental and storage conditions.

Problem Statement

Farmers often face crop spoilage due to high temperatures and inadequate storage facilities. The goal of this project is to identify spoilage risks early and support better decision-making.

Dataset

The project uses the Raksha agricultural dataset containing:

* Temperature
* Crop Type
* Location
* Storage Availability
* Spoilage Observed

Machine Learning Approach

* Learning Type: Supervised Learning
* Algorithm: Random Forest Classifier

Features Used

* Temperature_C
* Crop_Type
* Location
* Storage_Available
* Relative_Humidity_%
* Heatwave_Alert
* Days_In_Storage
* Transport_Delay_Hours

Output

The model predicts whether spoilage is likely to occur and helps reduce agricultural losses.
